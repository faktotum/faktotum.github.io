# Welcome to my Github Page

A smiley, sometimes referred to as a smiley face, is a basic ideogram that represents a smiling face, which has become part of popular culture worldwide. In modern times, the smiley has mostly been known for its yellow face.

## Character Traits

* Yellow
* Smiles
* Round
